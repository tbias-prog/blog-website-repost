# Editorial Blog Platform

Welcome to the **Editorial Blog Platform**, a modern, full-featured web application for creating, managing, and reading beautifully crafted stories.

---

## ✨ Full Features

Our platform provides everything you need to run an engaging editorial blog:

- **Dynamic Homepage:** View the latest published stories with an elegant, responsive grid layout. Let users easily discover your content.
- **Admin Dashboard:** Secure interface for creating, editing, and managing posts. Restricted strictly to authorized staff.
- **Rich User Authentication:** Sign up/Log in securely using Google, GitHub, or Email via Firebase Authentication.
- **Post Interaction:** Read detailed posts, interact with content, and leave feedback via our Contact page.
- **Cloud-Powered Backend:** Backed by Google Firebase (Hosting, Firestore, and Authentication) to ensure blazing-fast text delivery and reliable uptime.
- **Responsive Design:** Looks great on desktops, tablets, and mobile devices thanks to Tailwind CSS.

## 🛠️ Run Locally

**Prerequisites:** Node.js

1. Clone this repository:
   ```bash
   git clone https://github.com/tbias-prog/blog-website-repost.git
   cd editorial-blog-platform
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
